twinject (netdex)
=========
https://github.com/netdex/twinject

A bot for Touhou games that reads data directly from memory and uses this information in an attempt to bot the game. 
Please read all documentation in the repository.

-- INSTALLATION --
Extract all files to your Touhou game directory.

-- RUNNING --
Run twinject.exe, which will chainload the game.
Make sure your game executable and the one defined in twinject.ini match.

-- CONTROLS --
G - Enable bot
B - Disable bot
H - Display debug graphics
N - Hide debug graphics 
/ - Open debug console
